My version of the game MineSweeper.
Features:
* 3 Difficulty levels - Rookie / Sergeant / Monster
* 3 Safe clicks - shows a non-mine cell.
* 3 Hints - Show for a second a clicked cell and its neighbors.
* 3 Lives (Rookie has 2).
* High score.
